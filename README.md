# 图书管理系统 python15

## 项目概述

本图书管理系统基于Python语言和Flask框架，提供了一套可运行的源码，支持二次开发。

## 技术栈

- 后端：Python Flask
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

- **用户管理**
  - 登录

- **图书信息管理**
  - 添加
  - 修改
  - 查询
  - 删除
- **图书借阅记录**
  - 查询
- **读者信息管理**
  - 查询
